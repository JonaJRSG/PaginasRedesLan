Â¿Que es linux?

Linux es un "sistema operativo" de codigo abierto, lo que significa que todos pueden ver el codigo fuente y manipularlo segun lo que ellos busquen o quieran Eso lo vuelve super personalizable pero al mismo tiempo hace que sea mas complicado que otros sistemas operativos.

Tenemos que tener en cuenta que se le llama "Linux" a los sistemas operativos basados en el, aun que Linux realmente es el kernel. El kernel es el motor de los sitemas operativos, o para ser mas especificos es lo que une al software con el hardware, y la union de varios programas con el kernel es lo que conformaria un sistema operativo, pero como ya mencionamos Linux es de codigo abierto, por lo que mucha gente, grupos y empresas an creado distintos sistemas operativos con el kernel de Linux, a estos diferentes sistemas operativos se le llaman "Distribuciones" o "Distros" para los amigos, si quieres conocer mas de la historia de Linux preciona aqui o en la imagen de Linux
